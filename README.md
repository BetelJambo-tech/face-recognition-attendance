# 📸 Face Recognition Attendance System

A real-time face recognition system that automatically detects and records attendance using AI.

## 🚀 Features
- Face detection and recognition using OpenCV
- Multi-user support
- Automatic attendance logging
- Interactive dashboard with filtering (name, date)
- Upload image or use camera
- Download attendance records

## 🛠️ Tech Stack
- Python
- OpenCV
- Streamlit
- Pandas

## 💡 How It Works
1. User uploads image or uses camera
2. System detects face
3. Matches with trained data
4. Records attendance automatically
